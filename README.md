Mobilhaz — Netlify deploy

- Nyisd meg a Netlify weboldalát: https://app.netlify.com
- Sites → "Add new site" → "Deploy manually" → húzd ide a fájlt: `mobilhaz-netlify.zip` (a ZIP a projekt mappájának szülőkönyvtárában található).
- Szükség esetén állítsd be a site nevet és domain-t a Netlify felületén.

Szerkesztés menete:
- A weboldal HTML fájlját (`index.html`) szerkesztheted helyben. Ha többször szeretnéd frissíteni a publikált verziót, töltsd fel újra a frissített ZIP-et a Netlify felületére, vagy használd a Git + Netlify automatikus deployt.

Megjegyzés:
- A kontakt űrlap Formspree-hoz csatlakozik. A cél-email (forwarding) beállítását a Formspree fiókban ellenőrizd, ha nem érkeznének meg az üzenetek.
